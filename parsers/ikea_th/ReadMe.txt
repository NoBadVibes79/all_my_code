Это скрипт парсера для сайта Тайской Икеи - https://www.ikea.com/th/th/. 

Для того чтобы запустить скрипт:
1. Скачайте код себе любым удобным способом
2. Откройте файл parser_ikea_nsm.py в редакторе или блакноте
3. В консоли/командной строке необходимо установить библиотеки, ```pip install selenium, translate, bs4, pandas, requests, openpyxl``` - команда для установки через pip
4. Дополнительно к библиотеке selenium нужно установить драйвер хрома. 
  4.1 Проверить версию своего хрома (если нет установленного, то нужно скачать)
  4.2 Скачать хром драйвер такой же версии на этом сайте - https://chromedriver.storage.googleapis.com/index.html (у меня 114.0.5735.134, на нем всё работает стабильно)
    p.s. С версии 115+ - https://googlechromelabs.github.io/chrome-for-testing/
  4.3 Вы можете распаковать в папку со скриптом, тогда путь менять не надо внутри. Если хотите расположить его в другом месте, то распакуйте драйвер и измените 25 строку ```service = Service('chromedriver.exe')```, вместо chromedriver.exe, поместите путь до файла
  4.4 В строке 26 необходимо установить путь до файла, который запускает гугл хром - chrome.exe. На винде он находится - D:\Program Files\Google\Chrome\Application\chrome.exe
5. Скрипт принимает файл urls_category.txt, он должен находиться в одной директории со скриптом. 
Этот файл - это текстовый документ, в котором с каждой новой строки ссылка категории, с которых начнется сбор карточек товара.
Пример такого файла помещен в репозиторий с таким же именем
6. Запустите скрипт из редактора или через консоль. У меня это команда - python parser_ikea_nsm.py

При работе скрипт создает 3 файла. 
  1 - это index.html со всеми товарами, после того как скрипт их откроет на странице
  2 - Список url.txt товаров, с которых в циле будет собираться информация. Как раз можно проверить 
  3 - Результат - result.xslx - шаблон для удобной загрузки
